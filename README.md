# FoodVitals - AI Nutrition Tracker

AI-powered meal tracking and nutrition analysis app. Log meals, get nutritional breakdowns, and track dietary goals.

**Stack:** React Native, Expo, Supabase, TypeScript

## Features

- **Meal Logging** - Quick food entry with AI recognition
- **Nutrition Analysis** - Macro and micronutrient breakdowns
- **Goal Tracking** - Daily calorie and nutrient targets
- **History & Trends** - Visualize eating patterns over time
